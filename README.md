#BACKEND

1° Passo:
Fazer o git clone do projeto.
    Ex: git clone url do projeto
2° Passo:
Abrir o projeto em uma IDLE de preferência o VS CODE.
3° Passo: 
Instalar o node_modules com os seguintes comandos:
    npm install
4° Passo:  Executar o seguinte comando
    npm start.
